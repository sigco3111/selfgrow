# 자가발전 문명 시뮬레이션 (Self-Growing Civilization Simulation)

**순수 규칙 기반, LLM 없는 emergent behavior 관찰**

40마리의 개체가 유전 알고리즘과 시장 메커니즘, 유한 상태 기계, 그리고 스마트한 로컬 의사결정 엔진만으로 스스로 발전하는 문명을 만들어가는 시뮬레이션입니다. 외부 AI/LLM 의존성 없이 순수 로직만으로 emergent behavior(창발 행동)를 관찰하는 것이 목표입니다. 이 프로젝트는 **영원히 LLM을 연결하지 않습니다**.

---

## 주요 기능

### 🧠 듀얼 브레인 시스템 (Dual Brain System)
- **RuleBasedBrain** — 기존 행동 점수 기반 FSM (일반 개체, 기본값)
- **SmartBrain** — 경험 기반 학습 + 멀티스텝 계획 + 목표 설정 + 개체 간 메시징 (LLM 없음, 로컬 알고리즘)
- **비용 0**: 외부 API, LLM 호출 없이 순수 로컬 연산만으로 동작
- 두 브레인 타입 간 성능/생존율/부 축적 차이를 비교 관찰

### 🧬 유전 알고리즘 (Genetic Algorithm)
- **10가지 형질**: 위험감수, 호기심, 사회성, 공격성, 생산성, 혁신성, 힘, 체력, 민첩, 생식력, 충성심 (모두 0.0~1.0 정규화)
- **7가지 직업 특화**: 일반, 농부, 광부, 상인, 전사, 장인, 탐험가
- **유성/무성 생식**: 교차(Uniform Crossover) + 돌연변이(Mutation)로 자손 세대 형성
- **자연 선택**: 환경 적합도에 따른 개체의 생존/도태

### 🌍 월드 & 자원 시스템
- **40×30 타일 월드**, 7가지 지형 (평지, 숲, 산, 물, 사막, 언덕, 늪)
- **5가지 자원**: 식량, 목재, 돌, 철, 금 — 지형별로 분포 상이
- **자원 재생성**: 매 틱마다 최대치의 0.04%씩 회복 (희소성 유지)
- **타일별 편차**: ±50% 자원량 변동으로 부유한 땅과 빈약한 땅 구분

### 🏪 시장 경제 (Market Economy)
- **지정가 주문장**: 매수/매도 주문을 가격 우선으로 정렬하여 자동 체결
- **가격 발견**: 실시간 가격 지수 — 수요/공급에 따른 자연스러운 가격 변동
- **거래 수수료 2%**: 시장 활동에 세금 부과, 경제 지표 측정
- **주문 만료**: 20틱 후 자동 소멸, 유동성 유지

### ⚔️ 전투 & 영토 시스템
- **전투 메커니즘**: 공격력/방어력 기반 데미지 계산, 무기 장비 보너스 적용
- **영토 방어**: 본거지 근처에서 30% 전투 보정
- **약탈**: 전투 승자가 패자 인벤토리의 60% 획득
- **영토 클레임**: 일정 틱 이상 머문 타일을 자동으로 영토로 선언

### 🔧 제작 (Crafting)
- **5가지 레시피**: 돌도끼, 철검, 철갑옷, 낚싯대, 금장식
- **장비 효과**: 도구는 채집 보너스, 무기는 공격력, 갑옷은 방어력 증가
- **기술 의존성**: 철제 레시피는 야금술(Metallurgy) 선행 필요

### 📚 기술 트리 (Technology Tree)
- **9가지 기술**: 농업기초 → 관개, 채광 → 야금술 → 연금술, 화폐 → 관료제 등
- **문화적 진화**: 인접 개체 간 지식 전수 (사회성이 높을수록 더 많이 공유)
- **글로벌 연구**: 전체 개체의 혁신성과 지식 수에 비례해 연구 포인트 축적

### 📊 통계 지표
- **지니계수**: 경제 불평등 측정 (0=완전평등, 1=완전불평등)
- **분업지수**: 허핀달-허쉬만 지수 기반 직업 다양성
- **실시간 가격 추적**: 5가지 자원의 최근 100틱 가격 히스토리

### 🍃 계절 시스템 (Season System)
- **4계절 순환**: 봄/여름/가을/겨울 (각 25틱)
- **계절별 효과**: 자원 재생률, 에너지 소비, 채집 효율, 이동 속도 변동
- **봄**: 자원 재생 1.2배
- **여름**: 에너지 소비 1.15배
- **가을**: 채집 보너스 1.1배
- **겨울**: 이동 속도 0.8배, 에너지 소비 1.2배

### 🏗️ 건물 시스템 (Building System)
- **5가지 건물**: 저장고, 감시탑, 벽, 대장간, 제단
- **저장고**: 인벤토리 +5칸, 식량 저장 한도 +10
- **감시탑**: 탐험 범위 +2, 주변 개체 탐지 반경 +1
- **벽**: 영토 전투 시 방어력 +30%
- **대장간**: 제작 효율 +50%, 철 레시피 비용 -1
- **제단**: 사회성 +0.2, 지식 전수율 +20%

### 🎲 랜덤 이벤트 시스템 (Random Event System)
- **매 틱 2% 확률**로 이벤트 발생
- **최대 3개 동시 활성화**
- **이벤트 유형**: 자연재해, 풍년, 기근, 전염병 등
- **반경 및 지속시간**: 3~8타일, 10~25틱

### 🧠 이데올로기 시스템 (Ideology System)
- **4가지 이데올로기**: 물질주의, 군국주의, 영적주의, 평등주의
- **행동 바이어스**: 이데올로기에 따른 행동 선호도 변동
- **전파 메커니즘**: 인접 개체 간 이데올로기 전수
- **형성 조건**: 개체 나이가 30틱 이상일 때 유전자 형질에 따라 결정

## 시스템 구조

```
selfgrow/
├── requirements.txt      # 의존성 (Phase 0: 표준 라이브러리만)
├── .gitignore
├── README.md
└── sim/
    ├── __init__.py        # 패키지 초기화 — 주요 클래스 익스포트
    ├── config.py          # 모든 시뮬레이션 파라미터 (튜너블 상수)
    ├── genome.py          # 유전자 — 형질 정의, 돌연변이, 교차
    ├── entity.py          # 개체 — 상태 기계, 행동 실행 (두뇌는 brain.py에 위임)
    ├── brain.py           # 두뇌 시스템 — RuleBasedBrain / SmartBrain
    ├── knowledge.py       # 지식/기술 시스템 — 기술 트리, 개인 지식
    ├── resource.py        # 자원 — 자원 타입, 지형, 타일
    ├── world.py           # 월드 — 2D 격자, 공간 질의, 영토
    ├── market.py          # 시장 — 지정가 주문장, 체결, 가격 지수
    ├── metrics.py         # 통계 — 지니계수, 분업지수, 시계열 스냅샷
    ├── engine.py          # 엔진 — 메인 루프, 개체 생명주기, 이벤트
    ├── visualizer.py      # (현행) 터미널 시각화 — ANSI 이스케이프 코드
    └── main.py            # CLI 진입점 — 실행 인자 파싱, 메인 루프
```

## 설치 및 실행

### 요구 사항
- **Python 3.11+** (표준 라이브러리만 사용, 외부 패키지 불필요)

### 설치
```bash
# 저장소 클론
git clone https://github.com/sigco3111/selfgrow.git
cd selfgrow

# (선택) 가상 환경 생성
python -m venv .venv
source .venv/bin/activate  # Linux/Mac
.venv\Scripts\activate     # Windows
```

### 실행
```bash
# 기본 실행 (500틱, 실시간 시각화)
python -m sim.main

# 1000틱 실행
python -m sim.main --ticks 1000

# 빠르게 실행 (시각화 간격 늘리기)
python -m sim.main --interval 20

# 최대 속도로 실행 (시각화 없이, 종료 후 통계만 출력)
python -m sim.main --no-visual

# 속도 조절 (0.5 = 절반 속도, 0 = 최대 속도)
python -m sim.main --speed 0.5

# 특정 시드로 실행 (재현성)
python -m sim.main --seed 12345
```

### CLI 인자
| 인자 | 기본값 | 설명 |
|------|--------|------|
| `--ticks` | 500 | 최대 실행 틱 수 |
| `--seed` | 42 | 난수 시드 (재현성) |
| `--visual` | True | 실시간 시각화 (기본 켜짐) |
| `--no-visual` | False | 시각화 없이 실행 (헤드리스) |
| `--speed` | 1.0 | 시각화 속도 (0 = 최대 속도) |
| `--interval` | 5 | 화면 갱신 간격 (틱) |

## 시뮬레이션 메커니즘

### 개체 행동 결정 (Entity Decision Engine)
매 틱마다 각 개체는 다음과 같이 행동 점수를 계산하고 최고 점수의 행동을 수행합니다:

1. **CONSUME** — 배고픔 비율 > 20%이고 식량이 있으면 섭취
2. **REPRODUCE** — 에너지 충분, 번식 쿨다운 완료, 식량 충분 시 번식 시도
3. **GATHER** — 인벤토리에 공간이 있고 현재 타일에 자원이 있으면 채집
4. **TRADE** — 사회성에 비례해 잉여 자원 거래
5. **CRAFT** — 생산성이 높고 재료가 충분하면 도구/무기 제작
6. **COMBAT** — 식량 부족 시 약탈, 영토 방어, 일반 공격 순으로 전투 점수 계산
7. **EXPLORE** — 호기심에 비례해 미방문 타일 탐험
8. **IDLE** — 기본 대기

### 스마트 의사결정 (SmartBrain)
- **LLM을 사용하지 않습니다.** 개체의 뇌는 순수 로컬 알고리즘으로 동작합니다.
- SmartBrain 개체는 과거 경험을 기억하고, 멀티스텝 계획을 세우며, 목표를 설정하고, 다른 개체와 메시지를 주고받습니다.
- SmartBrain의 emergent behavior가 RuleBasedBrain과 어떻게 다른지 비교 관찰합니다.

### Emergent Behavior 기대 효과
- **경제 계층 분화**: 환경 적응에 따른 농부/광부/상인 등 직업 분화
- **시장 가격 변동**: 자원 희소성에 따른 자연스러운 가격 발견
- **영토 분쟁**: 자원이 풍부한 지역을 둘러싼 개체 간 갈등
- **문화적 진화**: 인접 개체 간 지식 전수를 통한 기술 확산
- **기술 격차**: 기술 선점 개체와 그렇지 못한 개체 간 생산성 차별화

## 개발 진행 상황

### Phase 0 — 코어 엔진 ✅
- [x] 유전 알고리즘 (돌연변이, 교차, 자연 선택)
- [x] 2D 월드와 7가지 지형
- [x] 9개 노드 기술 트리
- [x] 지정가 주문장 시장 시스템
- [x] 전투/영토 약탈 메커니즘
- [x] 제작 시스템 (5종 레시피)
- [x] 경제/인구 통계 (지니계수, 분업지수)
- [x] 문화적 진화 (지식 전수)
- [x] 사이버펑크 TUI (Rich)

### Phase 1 — 밸런싱 + 전투 활성화 ✅
- [x] 자원 분포 불균형 (타일 편차 ±50%)
- [x] 영토 클레임 시스템 (본거지 + 반경 방어)
- [x] 약탈 트리거 (식량 부족 시 이웃 공격)
- [x] 영토 방어 (본거지 근처 30% 전투 보정)
- [x] 가격 바닥제 + 인구 밸런싱

### Phase 2 — 전투 심화 + 파벌 ✅
- [x] 파벌 시스템 (자동 결성/해체, 지도자, 영토)
- [x] 다중 개체 전투 (동맹 지원)
- [x] 장비 파괴/약탈 (8% 파괴, 20% 약탈)
- [x] 지식 약탈 (전사한 적의 기술 30% 확률로 획득)
- [x] 후퇴 로직, 파벌 전쟁

### Phase 3 — 스마트 의사결정 ✅
- [x] **SmartBrain**: 경험 기반 학습 — 과거 행동 결과를 기억해 더 나은 선택
- [x] **멀티스텝 계획**: 2~3틱 앞을 내다보는 행동 선택
- [x] **목표 설정**: 단기 목표(자원 수집, 기술 습득) 자동 설정 및 추적
- [x] **개체 간 메시징**: 구조화된 메시지로 거래/동맹 협상 (LLM 없음)
- [x] **실험 설계**: Smart 개체 vs 일반 개체 생존율/부 축적 비교 (실시간 TUI + 최종 요약)

> **⚠️ LLM 관련 공지**: 이 프로젝트는 앞으로도 **영원히 LLM/외부 AI를 연결하지 않습니다**.
> 모든 의사결정은 순수 로컬 알고리즘(SmartBrain)으로 처리합니다.
